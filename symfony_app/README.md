# Mixas Art Works

Сайт-портфолио работ по покрасу миниатюр.

## Что внутри

- Публичные страницы:
  - Главная
  - Контакты
  - Фотогалерея
  - Обратная связь
- Раздел работ:
  - у каждой работы отдельная страница;
  - у каждой работы можно хранить несколько фотографий.
- Админ-раздел:
  - управление страницами;
  - управление работами;
  - управление фотографиями работ.

## Запуск

```bash
docker-compose up --build -d
```

Сайт: `http://localhost:8080`

При первом запуске контейнер `php` автоматически:
- применяет миграции;
- загружает стартовые данные.

Изменения в `src/`, `templates/`, `assets/`, `public/` применяются без перезапуска контейнеров.

## Composer в контейнере

Composer установлен в `php`-контейнере:

```bash
docker-compose exec -T php composer --version
docker-compose exec -T php composer install
```

## Выбор фото из R2 в админке

В форме создания/редактирования работы есть поле `Фото из R2 для этой работы`.
После сохранения выбранные пути автоматически добавляются как фото работы.

Чтобы включить список файлов из R2, задай переменные окружения:

```dotenv
R2_ENDPOINT=https://<accountid>.r2.cloudflarestorage.com
R2_BUCKET=<bucket-name>
R2_ACCESS_KEY_ID=<access-key-id>
R2_SECRET_ACCESS_KEY=<secret-access-key>
R2_REGION=auto
R2_IMAGES_PREFIX=mixas-works/
R2_LIST_LIMIT=500
```

Если credentials не заданы, форма остаётся рабочей, но выбор из хранилища отключен.
