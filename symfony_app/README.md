# Mixas Art Works

Сайт-портфолио работ по покрасу миниатюр.

## Что внутри

- Публичные страницы:
  - Главная
  - Контакты
  - Фотогалерея
  - Обратная связь
- Раздел работ:
  - у каждой работы отдельная страница;
  - у каждой работы можно хранить несколько фотографий.
- Админ-раздел:
  - управление страницами;
  - управление работами;
  - управление фотографиями работ.

## Запуск

```bash
docker-compose up --build -d
```

Сайт: `http://localhost:8080`

При первом запуске контейнер `php` автоматически:
- применяет миграции;
- загружает стартовые данные.

Изменения в `src/`, `templates/`, `assets/`, `public/` применяются без перезапуска контейнеров.

## Composer в контейнере

Composer установлен в `php`-контейнере:

```bash
docker-compose exec -T php composer --version
docker-compose exec -T php composer install
```
