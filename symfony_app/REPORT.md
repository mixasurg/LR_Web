# Mixas Art Works

## Описание проекта

Mixas Art Works — сайт-портфолио работ по покрасу миниатюр.

## Страницы сайта

- Главная
- Контакты
- Фотогалерея
- Обратная связь
- Отдельные страницы работ (с несколькими фотографиями)

## Контент

- Текстовые описания работ
- Таблица контактов
- Фотогалерея
- Видео-блок на главной

## Админ-раздел

- Управление страницами
- Управление работами
- Управление фотографиями работ

## База данных (кратко)

- Пользователи
- Страницы
- Работы
- Фотографии работ
- Сообщения обратной связи

## Скриншоты для отчета

- Главная страница
- Контакты
- Галерея
- Страница отдельной работы с несколькими фото
- Обратная связь
- Админ: страницы / работы / фото
